# [项目]

## Amphor Protocol DeFi dApp

**项目类型：** 面向 Ethereum 的 DeFi 收益聚合协议应用  

**项目角色：** 全栈工程师  

围绕 Boosted ETH Vault 与 Symbiotic LRT Vault 构建完整 dApp 闭环，覆盖 Vault 存取、Dashboard Claim、Points / Referral / Quest、多资产入金、链上状态聚合以及运营配置后台。

**核心职责：**

- 独立负责 dApp 主流程前端开发与 Node.js 后端架构设计，覆盖钱包连接、Earn / Vault Detail、Deposit / Withdraw、Dashboard、Claim、Points、Referral、Quest 等核心模块。
- 基于 Next.js、React、TypeScript、TanStack Query、Zustand 与 wagmi 实现应用前端，完成链上交互、状态展示与复杂业务流程编排。
- 基于 Node.js、NestJS、PostgreSQL、Prisma、Redis、BullMQ 搭建后端服务，处理链上状态聚合、奖励账本、运营配置及异步任务调度。

**技术难点：**

- 设计异步 Vault 多阶段交易状态流，统一 request → processing → bounding → settlement → dashboard claim 的业务状态表达，解决“链上确认不等于业务完成”的状态割裂问题。
- 支持 ETH、LST / LRT、稳定币等多资产入金路径，处理 Zapper 报价、滑点控制、授权路径、Permit 签名路径及异常回滚场景。
- 实现 Points / Rewards / Referral / Quest 多层奖励体系，保证账本、资格校验、展示状态与可领取状态一致。

**项目结果：**

- 完成主网版本发布，形成 Vault + Dashboard + Claim + Points + Quest 的完整产品闭环。
- 协议被 DefiLlama 收录，产品真实进入链上运营阶段，可公开查询 TVL、Fees、Revenue 等指标。
